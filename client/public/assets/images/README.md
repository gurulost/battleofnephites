# Game Image Assets

This directory contains SVG assets for units and buildings in the game.

Required SVG files:
- nephite-worker.svg
- nephite-melee.svg
- nephite-ranged.svg
- nephite-city.svg
- nephite-barracks.svg
- lamanite-worker.svg
- lamanite-melee.svg
- lamanite-ranged.svg
- lamanite-city.svg
- lamanite-barracks.svg

The game currently looks for these files, and will display errors if they are missing.
